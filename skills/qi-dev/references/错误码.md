# qi 框架预定义错误码（1000–1103）

> 业务错误码从 **2000+** 开始，避免冲突。

| 变量 | Code | HTTP Status | 默认消息 |
|------|------|-------------|----------|
| `qi.ErrServer` | 1000 | 500 | server error |
| `qi.ErrBadRequest` | 1001 | 400 | bad request |
| `qi.ErrUnauthorized` | 1002 | 401 | unauthorized |
| `qi.ErrForbidden` | 1003 | 403 | forbidden |
| `qi.ErrNotFound` | 1004 | 404 | not found |
| `qi.ErrConflict` | 1005 | 409 | conflict |
| `qi.ErrTooManyRequests` | 1006 | 429 | too many requests |
| `qi.ErrInvalidParams` | 1100 | 400 | invalid parameters |
| `qi.ErrMissingParams` | 1101 | 400 | missing parameters |
| `qi.ErrInvalidFormat` | 1102 | 400 | invalid format |
| `qi.ErrOutOfRange` | 1103 | 400 | out of range |

## 错误克隆链用法

```go
// 附加底层错误和自定义消息（返回新实例）
return nil, qi.ErrNotFound.
    WithErr(sql.ErrNoRows).
    WithMessage("用户 " + id + " 不存在")

// 仅修改消息
return nil, qi.ErrBadRequest.WithMessage("page 不能小于 1")

// 业务包自定义错误
var ErrOrderTimeout = errors.NewWithStatus(3001, http.StatusGatewayTimeout, "订单超时")
```
